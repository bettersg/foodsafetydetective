Wash Hand Game prototype

Game instruction

1) To play use the up and down arrows and 'enter' button to select

2) To interact press 'a' key

3) To acess menu press 'esc' key/button

- Once in menu select 'resume game' to go back to game

- If select instruction, press 'esc' to return to main menu

4) Follow the key stated [end] means press 'end' button etc..

5) Enjoy and have fun.
